# AI Works Portfolio

Claude Code を中核に構築したアプリ・スキル・Subagent の成果物カタログ。

🔗 **公開URL:** https://amberchai11.github.io/ai-works-portfolio/

## 概要

手を動かして検証した制作物だけを掲載したポートフォリオサイトです。

- **Apps (3):** Puyo Puyo ゲーム / 電卓モジュール / 英単語フラッシュカード
- **Skills (5):** file-summarize / blog-polish / char-count / review / seo-researcher
- **Subagents (7):** code-reviewer / doc-writer / test-runner / seo-blog-writer / draft-writer / fact-checker / editor-polisher

## 技術

- 依存ゼロの静的サイト（HTML / CSS / Vanilla JS）
- Inter（Google Fonts）によるサンセリフタイポグラフィ
- レスポンシブ対応（3カラム → 2カラム → 1カラム）
- スクロール連動のフェードイン（IntersectionObserver）／カテゴリフィルター

## デザイン

「専門性・信頼・国際性・透明性・洗練」をテーマにした、白基調＋近黒のコントラストにイエローを最小限のアクセントとして効かせたクリーンなデザイン。

## ローカル確認

```bash
python -m http.server 8000
# http://localhost:8000 を開く
```
